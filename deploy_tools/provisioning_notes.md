Provisioning a new site
=======================

## Required packages:

* Python 3
* Git (pre-installed on Dreamhost)
* pip (pre-installed '')
* virtualenv (pre-installed '')

##

* add shared libraries to LD_LIBRARY_PATH to fix 'not found' error





## Folder structure:
Assume we have a user account at /home/username

/home/username
└── sites
    └── SITENAME
         ├── database
         ├── source
         ├── static
         └── virtualenv
